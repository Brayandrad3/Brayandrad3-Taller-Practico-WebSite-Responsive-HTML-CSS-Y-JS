# 🌐 Responsive Website | HTML, CSS & JavaScript

Una página web responsiva desarrollada como parte de un taller práctico para fortalecer habilidades en desarrollo frontend utilizando **HTML5**, **CSS3** y **JavaScript**.

El proyecto implementa principios de diseño responsivo para ofrecer una experiencia de usuario consistente en diferentes tamaños de pantalla.

---

## 🚀 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## ✨ Características

- Diseño responsive (Mobile First)
- Estructura semántica con HTML5
- Estilos modernos con CSS3
- Interactividad con JavaScript
- Compatible con dispositivos móviles, tabletas y escritorio

## ▶️ Cómo ejecutar el proyecto

1. Clona este repositorio.

```bash
git clone https://github.com/Brayandrad3/Brayandrad3-Taller-Practico-WebSite-Responsive-HTML-CSS-Y-JS.git
```

2. Accede a la carpeta del proyecto.

```bash
cd Brayandrad3-Taller-Practico-WebSite-Responsive-HTML-CSS-Y-JS
```

3. Abre el archivo `index.html` en tu navegador o utiliza la extensión **Live Server** de Visual Studio Code.
